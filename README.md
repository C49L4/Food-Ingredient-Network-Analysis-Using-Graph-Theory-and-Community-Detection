# Food-Ingredient-Network-Analysis-Using-Graph-Theory-and-Community-Detection
Graph-based analysis of food ingredient relationships using network science.

This project presents a graph-based analysis of food ingredient relationships using network science techniques. The dataset is transformed into a structured network where ingredients are represented as nodes and their co-occurrence in recipes defines the edges between them. By applying graph theory methods, the study explores the underlying structure of culinary combinations and identifies meaningful patterns within ingredient interactions.

Community detection algorithms are used to reveal clusters of ingredients that frequently appear together, effectively uncovering hidden “flavor communities” within the dataset. In addition, centrality measures such as degree and betweenness are utilized to identify key ingredients that play a significant role in connecting different culinary groups.

The project emphasizes data preprocessing, network construction, and visualization techniques to transform raw culinary data into an interpretable network structure. The final output provides both static and interactive visualizations that help in understanding how ingredients are connected and how different flavor profiles emerge from these relationships.

This work demonstrates how graph theory can be applied beyond traditional domains to analyze culinary data and extract meaningful insights about food composition and structure.
